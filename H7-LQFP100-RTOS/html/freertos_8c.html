◆ AUTODEINIT_init()

void AUTODEINIT_init ( void * argument)

Initializes the AUTODEINIT thread.

This function waits for AUTODEINIT event flags, disables all voltages from 15V to SDN1, updates the rail monitor, and yields control to other threads in an infinite loop.

◆ AUTOINIT_init()

void AUTOINIT_init ( void * argument)

Initializes the AUTOINIT thread.

This function waits for AUTOINIT event flags, enables all voltages from SDN1 to 15V, updates the rail monitor, and yields control to other threads in an infinite loop.

◆ Idle_init()

void Idle_init ( void * argument)

Initializes the Idle thread.

This function waits for the IDLE event flag, disables ERPA and HK, stops the DAC, updates the rail monitor, disables all voltages, and yields control to other threads.

◆ Science_init()

void Science_init ( void * argument)

Initializes the Science thread.

This function waits for the SCIENCE event flag, enables all voltages, updates the rail monitor, starts DAC in DMA mode, and resets packet sequence numbers, transitioning to the science mode.

◆ STOP_init()

void STOP_init ( void * argument)

Initializes the STOP thread.

This function waits for the STOP event flag, clears it, sets the IDLE flag, and enters stop mode when idle.

◆ Sync_init()

void Sync_init ( void * argument)

Initializes the Sync thread.

This function waits for the SYNC event flag, sends an acknowledgment, waits for a specific key value to be received, calibrates the RTC, and handles UART communication for error counting and reset cause.

◆ Voltage_Monitor_init()

void Voltage_Monitor_init ( void * argument)

Initializes the Voltage Monitor thread.

This function waits for voltage monitor event flags, sets the rail monitor, and checks rail status. If rails are out of bounds, it transitions to idle mode before returning to science mode.
