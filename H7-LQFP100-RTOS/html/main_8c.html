◆ system_setup()

void system_setup ( )

Performs system setup tasks.

This function initializes various system components including timers, ADC calibration, and DMA for ADC data acquisition. It starts PWM for TIM2, performs ADC calibration for ADC1 and ADC3, and starts DMA for ADC data acquisition. Any errors encountered during these initialization steps are handled by the Error_Handler function.
