Sets the time for the RTC.

This function configures the time in the RTC peripheral. The time is set in 24-hour format.

Parameters
hrtcPointer to a RTC_HandleTypeDef structure that contains the configuration information for the specified RTC.
sTimePointer to a RTC_TimeTypeDef structure that contains the time values to be set.
FormatSpecifies the format of the time (24-hour or 12-hour).
Return values
HAL_StatusTypeDefHAL_OK on success, HAL_ERROR on failure.
